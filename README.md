# GDB utils
##### prerequisite
```
sudo apt-get install python3-pip
sudo apt-get install tk-dev
sudo pip install python3-tk // OR sudo apt install python3-tk
sudo pip3 install numpy
cp .gdbinit ~/.gdbinit
sudo pip3 uninstall -y matplotlib
sudo pip3 --no-cache-dir install -U matplotlib
sudo pip3 install pysigset
# in GDB command first you need call init_debug
```
